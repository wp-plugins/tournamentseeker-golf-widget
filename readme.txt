Showcase your favorite TournamentSeeker golf events in a stylish sidebar widget!

== Description ==

Enjoy finding or listing golf tournaments on http://golf.tournamentseeker.com/? This widget lets you showcase these events on your own site. 

With this widget, you have three different options for displaying events:
1. Display the events for which you have submitted listings
2. Display events that you have marked as "favorites"
3. Display a variety of events based on a search term of your choosing

Note: To use this widget, you will need to have an active account on http://golf.tournamentseeker.com/.

== Installation ==

1. Install and activate the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to http://golf.tournamentseeker.com/api, sign in, and grab your API credentials from the bottom of the page
4. Add the widget to your desired sidebar in the 'Widgets' menu in WordPress
5. Add your API credentials to the widget options, select your display options, and you're set!

== Frequently Asked Questions ==

= Where do I find my API key? =

After signing into the TournamentSeeker site, go to http://golf.tournamentseeker.com/api. At the bottom of the page, you should find a box containing your public key and secret key. These are the two pieces of your API key that you'll need to enter into the widget options in order for the TS system to know who you are and which events you've listed or favorited.

= Will this work for other TournamentSeeker sites? =

No, this plugin is specific to the golf listings featured on http://golf.tournamentseeker.com/. To display event listings from another TournamentSeeker site, please check our other offerings for the widget specific to that site.
